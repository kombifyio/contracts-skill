## Contracts Skill With Beads

Before changing code, run a Beads-enforced contract preflight. Use `bd list --status open --tag contracts` and create or reuse a preflight task with `bd create` when needed. Find the nearest `CONTRACT.md`, read it with `CONTRACT.yaml`, compare `meta.source_hash`, summarize relevant MUST/MUST NOT constraints, close the preflight task with the result, and sync YAML before implementation if drift exists.
