# Sample Project

An example project demonstrating the Contracts skill.

## Structure

```
sample-project/
├── .contracts/
│   └── registry.yaml           # Contract index
│
└── src/
    ├── core/
    │   └── auth/
    │       ├── CONTRACT.md     # User specification
    │       └── CONTRACT.yaml   # Technical spec
    │
    └── features/
        └── dashboard/
            ├── CONTRACT.md
            └── CONTRACT.yaml
```

## Usage

1. Open this folder in VS Code
2. Install the contracts skill
3. Run `check contracts` to verify everything is synced
